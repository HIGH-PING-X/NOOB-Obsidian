```java
final修饰的类无法继承
final修饰的方法无法覆盖
final修饰的变量只能赋值一次
final修饰的引用一旦指向某个对象则不能指向其他对象，但该引用指向的对象内部的数据是可以修改的
final修饰的实例变量必须手动初始化，不能采用系统默认值
final修饰的实例变量一般和static联合使用，成为常量
public static final String NAME = "NOOB" //修饰的是常量，值不可变，常量名大写
```
