| 元素     | 语法                                                   |
|:-------- |:------------------------------------------------------ |
| 标题     | `# H1`<br>`## H2`<br>`### H3`                          |
| 粗体     | `**bold text**`                                        |
| 斜体     | `*italicized text*`                                    |
| 引用快   | `> blockquote`                                         |
| 有序列表 | `1. First item`<br>`2. Second item`<br>`3. Third item` |
| 无序列表 | `- First item`<br>`- Second item`<br>`- Third item`    |
| 代码     | ``code``                                               |
| 分隔线   | `---`                                                  |
| 链接     | `[title](https://www.example.com)`                     |
| 图片     | `![alt text](image.jpg)`                               |
